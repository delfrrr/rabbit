# Carrot Bunnies Game

A platformer game for Rabbit R1 where you play as a bunny collecting carrots!

## How to Play

1. **Start the Game**: Click "Play" from the main menu, or select a level from "Level Select"

2. **Controls**:
   - **Keyboard**:
     - Arrow Left/Right: Move left/right
     - Space: Jump
     - P: Pause
   - **Touch Controls** (on-screen buttons):
     - ‚Üê : Move left

3. **Objective**:
   - Collect carrots (ü•ï) scattered throughout each level
   - Reach the green exit (E) with enough carrots
   - Each level requires a minimum number of carrots to complete

4. **Game Mechanics**:
   - Collect carrots: +1 carrot per carrot collected
   - Falling (off the map or hitting hazards): Lose 3 carrots
   - If you fall 3 times, the level fails
   - Brown blocks (#) are solid platforms you can stand on
   - Red spikes (^) are hazards - avoid them!

5. **Levels**:
   - Level 1: Tutorial level, requires 8 carrots
   - Level 2: Adds gaps, requires 10 carrots
   - Level 3: Adds moving obstacles, requires 12 carrots
   - Level 4: Tighter jumps, requires 14 carrots
   - Level 5: Final level with all challenges, requires 16 carrots

6. **HUD**: The top-right corner shows:
   - ü•ï Carrot count
   - Current level (Lvl X/5)
